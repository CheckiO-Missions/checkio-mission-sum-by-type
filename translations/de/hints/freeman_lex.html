Die allgemeine Idee ist, sich über die Liste zu bewegen und, je nach Art des aktuellen Wertes, diesen Wert an die Variable zu verketten oder hinzuzufügen.
Okay, beginnen wir mit der Implementierung in Code. Welches Instrument von Python sollte ich verwenden um über die Liste zu gehen?
In dieser Lösung biete ich Ihnen an, for loop zu verwenden: 
for value in items:
    # do something
Zuvor schrieben Sie 'abhängig vom Typ des aktuellen Wertes'. Könnten Sie diesen Schritt bitte präzisieren?
Ja, natürlich! Um zwischen verschiedenen Befehlen zu wählen, verwenden Sie if statement: 
if condition == True:
    # do something
else:
    # do something else
Und was ist meine Bedingung für die if-Anweisung?
Nun, hier brauchen Sie type() funktion und Typen von Eingabewerten in dieser Mission, die Sie bereits aus der Beschreibung kennen - str und int: 
if type(value) == str:
    # do something
else: # in case of int value
    # do something else
Da Sie in dieser Aufgabe nur zwei Arten von Werten in der Liste haben, brauchen Sie den anderen Typ nicht zu überprüfen.
Aber dies ist nur eine Prüfung. Ich habe das Gefühl, dass ich die vorherigen Hinweise kombinieren muss, aber ich weiß nicht, wie. Ich brauche also einen weiteren Hinweis.
Ja, Sie haben Recht! Um alle Werte zu überprüfen, müssen Sie die Überprüfung bei jeder Iteration der for-Anweisung überprüfen. Sie müssen also die if-Anweisung innerhalb der for-Anweisung einfügen: 
for value in items:
    if type(value) == str:
        # do something
    else:
        # do something else
Ok. Was ist als nächstes zu tun? Wie kann ich hinzufügen oder verketten?
Zuallererst müssen Sie vor der for-Anweisung zwei Variablen deklarieren, eine - für Ihre zukünftige verkettete Zeichenkette, die andere - für die Summe der ganzen Zahlen. 
conc = ''
sumed = 0
Dann führen Sie innerhalb der if- und else-Anweisungen die entsprechenden Operationen durch. Beide Typen unterstützen den Verkettungsoperator '+='. Aktualisieren Sie vorher deklarierte Variablen: 
conc = ''
sumed = 0
for value in items:
    if type(value) == str:
        conc += value
    else:
        sumed += value
Es fehlt noch etwas. Was ist die endgültige Lösung?
Ja, Sie müssen die Funktion selbst und die Rückgabewerte deklarieren die in conc und summierten Variablen als Tupel gespeichert sind: 
def sum_by_types(items: list[str, int]) -> tuple[str, int] | list[str, int]:
    
    conc = ''
    sumed = 0
    for value in items:
        if type(value) == str:
            conc += value
        else:
            sumed += value
                    
    return (conc, sumed)
Ich frage mich, ob diese Aufgabe in einer Zeile lösbar ist?
Ja, denn Python ist sehr cool! Einer der so genannten Oneliner ist der folgende und muss zusätzlich importiert werden: 
from functools import reduce

def sum_by_types(items: list[str, int]) -> tuple[str, int] | list[str, int]:
    
    return reduce(lambda a, b: (a[0] + b, a[1])
                               if type(b)!= int
                               else (a[0], a[1] + b),
                               [('', 0)] + items)
