Something is still missing. What is the final solution?
Yes, you must declare the function itself and return values stored in conc and sumed variables as tuple: 
def sum_by_types(items: list[str, int]) -> tuple[str, int] | list[str, int]:
    
    conc = ''
    sumed = 0
    for value in items:
        if type(value) == str:
            conc += value
        else:
            sumed += value
                    
    return (conc, sumed)
I wonder, if this mission is solvable in one line?
Yes, because Python is very cool! One of so called oneliners is the following and needs additional import: 
from functools import reduce

def sum_by_types(items: list[str, int]) -> tuple[str, int] | list[str, int]:
    
    return reduce(lambda a, b: (a[0] + b, a[1])
                               if type(b) != int
                               else (a[0], a[1] + b),
                               [('', 0)] + items)
